K线数据

不同周期的数据放到对应的目录下面
目录名字对应周期 0=日线 1=周线 2=月线 3=年线 4=1分钟 5=5分钟 6=15分钟 7=30分钟 8=60分钟 9=季线 10=分笔数据

0=日期 格式YYYYMMDD 如20190229=>2019-02-29
1=前收盘
2=开盘价
3=最高
4= 最低
5=收盘
6=成交量 (单位股)
7=成交金额
8=时间格式 hhmm 如(925=>9:25)  日线没有这个数据(可以填0)
9=涨停家数
10=跌停家数



分笔数据格式
0=日期 格式YYYYMMDD 如20190229=>2019-02-29
1=时间 格式 hhmmss
2=价格
3=成交量
4=成交金额
5=B/S  1=B 2=S 0=不明盘
